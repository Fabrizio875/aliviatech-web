# AliviaTech — Web (Landing + Dashboard)

Estructura profesional tipo repositorio para despliegue rápido (Netlify / GitHub Pages).

## Scripts
- `npm start` — inicia un servidor local si tienes `live-server` instalado globalmente.

## Estructura
- `index.html` — landing page en español (nombres técnicos en inglés).
- `pages/dashboard.html` — panel web en español.
- `css/styles.css`, `js/main.js` — estilos y scripts.
- `api/getMoodData.js` — API simulada en el cliente.
- `docs/` — documentación para Tasayco y Yerico.
